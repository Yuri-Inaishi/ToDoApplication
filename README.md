# ToDoApp

これは、HTML、CSS、JavaScriptで作成された、シンプルで直感的なToDoリストアプリケーションです。

## 概要

プラント設計エンジニアからWeb開発者へのキャリアチェンジを目指し、制作したToDo管理アプリケーションです。日々の業務やプライベートで活用しています。
